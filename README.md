# brew-log

**Version 0.3.0**

Log your coffee/matcha/tea intake — what, when, and where (optional per-brew location with recent-place suggestions). Edit any brew's date, rating, notes, name, or type after the fact, and browse a calendar to see past days' brews and the "daily cup" mug for that day. Uses local storage on your phone. Designed as a web app, intended to be added as a PWA on iPhone.

Curious without signing in? Tap **"Explore the demo"** on the login screen for a pre-seeded, local-only walkthrough — no Google account, nothing synced.

Built with React + Vite.
